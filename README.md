# Ecommerce-Sales-Analysis
Data analysis project using Google Sheets and Looker Studio.
## Problem Statement
My goal is to conduct a regional, categorical, and sales analysis of an e-commerce website.
## Data Source 
[GoogleSheets](https://docs.google.com/spreadsheets/d/1llrvzpa3x50fLIxfxqos40gXzgV9VQNnEnPazcXmiL0/edit?gid=452170247#gid=452170247)
[LookerStudio](https://lookerstudio.google.com/reporting/ea7da149-0398-4d89-a751-9cee1cb27f13) 
ecommerce_sales_data RawData.csv
## Data Cleaning & Transformation
I imported the data from the CSV file into Google Sheets. I added cost per item and price columns to the Discovery section. I changed the formatting of the necessary columns. I imported this file into Looker Studio. I created the Dashboard and Results sections as three separate pages.
